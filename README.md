# Kaharagia Governance Manual

This repository contains the Kaharagia governance and court manual in Markdown. All published sections live under `vol1/`, with a linked table of contents in `vol1/README.md`.

## Contents
- `vol1/` — Volume I manual sections and the government structure overview, each with YAML metadata for doc-site ingestion
- `vol1/README.md` — index linking every section in order

## Usage
- Browse or render the Markdown directly.
- Doc-site import: point your static site generator at `vol1/` and use the `order` metadata for navigation.
